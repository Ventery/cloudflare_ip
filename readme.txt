胡言乱语
翻墙被封是常态了，不过也不用担心，gfw当然没有能力破解你的加密内容，它只是硬封罢了，顶多识别一下流量特征，只要你不在外网泄露太多个人信息，也不在本地使用某些设备和软件那就没事（当然了最好别做坏事）
通常解决方式是升级技术，但是偶尔也还是会被封，换域名的话也很麻烦，治标不治本
所以比较好的解决方式是换ip

用cloudflare代理你的翻墙工具是个不错的选择，相当于就换了个ip（自己服务器ip不变，中转变了），我上一个ip被封也是去年的事了，被封的不算太频繁
只是在大陆地区当地的dns解析的cf的ip也常常遭到封禁
具体有两种封禁方式
第一：tcp连接超时（可以研究下握手细节，肯定是是gfw从中作梗）
第二：tls握手失败（理由同上）

虽然本地dns解析的cf的ip是被封禁的，但是cf拥有的ip很多，我们可以手动修改，找到一个可用的ip还不是难事
具体是通过修改host中的解析来实现，把你的域名写成cf的ip就行
cf的ip段是在官网中公开的

也不用担心cf的所有ip被封禁，我估计可能因为他们也在用
